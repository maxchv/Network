
К серверу написанному в последнем домашнем задании
добавить функцию получения данных передаваемых
пользователем в запросе по методам GET и POST.

Реализовать веб-приложение которое использует созданный сервер.
Назначение веб-приложения - работа с данными сотрудников
некой коммерческой компании (для отдела кардров).
В задачи входит:
	1. Добавление сотрудника
	2. Удаление сотрудника
	3. Поиск сотрудника по фамилии
	4. Отображение всех сотрудников
	5. Редактирование данных сотрудника

У сотрудника есть следующие данные:
	0. Id
	1. ФИО
	2. Возраст
	3. Зарплата
	4. Должность

При выставлении оценки будет оцениваться удобство пользовательского
интерфейса, а также архитектура сервера и веб-приложения.

Имейте ввиду, что сервер - один, а веб приложений может быть много.
Поэтому разработайте такую архитектуру сервера что бы на нем можно
было без существенных изменений запустить другое веб-приложение
(в идеале в сервер не должно быть внесено ни единой строки кода,
максимум - конфигурирование через файл app.config)
